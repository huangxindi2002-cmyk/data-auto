❓ 怎么生成 PAT？怎么配置？ 
1. 登录 GitHub → 右上头像 → Settings
2. 左侧 Developer settings → Personal access tokens → Tokens (classic)
3. Generate new token (classic)
   - Note: data-auto
   - Expiration: 90 天
   - Scopes 勾选: ☑ repo  ☑ workflow
4. 生成后复制 ghp_xxxxxxxx，粘贴到上面 PAT 输入框
5. 仓库 Settings → Secrets and variables → Actions
   → New repository secret: 名字 DATAAI_API_KEY, 值 = data.ai 的 key
